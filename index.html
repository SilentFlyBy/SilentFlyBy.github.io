Statische Webseiten müssen nicht schwer sein.

March 26 2019 | programming | #web

Wer Webseiten mit einheitlichem Look bauen möchte, stößt mit reinem HTML schnell an seine Grenzen. Webseiten bestehen meist aus einem einheitlichen Header, Footer und vielleicht einer Sidebar. Nur der Content dazwischen unterscheidet sich von Seite zu Seite. Versucht man nun, eine mehrere Unterseiten umfassende Webseite nur aus HTML-Dateien zu erstellen, muss man Header, Footer und Sidebar auf jeder Seite manuell einfügen. Ändert sich der Code an einem dieser Elemente, muss man die Änderung auf jeder einzelnen Unterseite mitziehen. ...
